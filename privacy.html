Kebijakan Privasi – CatatBisnis

Terakhir diperbarui: 17 Oktober 2025

Aplikasi CatatBisnis dikembangkan oleh Tri Agung Biantoro. Kebijakan privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pengguna.

1. Informasi yang Kami Kumpulkan

  • Informasi akun pengguna (nama, email, nomor telepon jika ada akun dibuat)
  • Lokasi perangkat (untuk fitur absensi berbasis lokasi)
  • Data penggunaan aplikasi (log aktivitas, versi aplikasi, jenis perangkat)

2. Penggunaan Data

Data digunakan untuk menyediakan layanan, menyimpan catatan transaksi, dan memberikan dukungan teknis.

3. Penyimpanan dan Keamanan

Data dikirim melalui HTTPS dan disimpan di server aman. Kami tidak membagikan data kepada pihak ketiga tanpa izin pengguna.

4. Izin Aplikasi

  • Lokasi — untuk mencatat lokasi absensi pengguna
  • Penyimpanan — untuk menyimpan file laporan
  • Internet — untuk sinkronisasi data

5. Anak-Anak

Aplikasi ini tidak ditujukan untuk anak di bawah 13 tahun.
